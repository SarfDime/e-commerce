## Installation

```bash
$ npm install
```

## Running the app

-- GraphQL Playground available at http://localhost:3000/graphql 

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
